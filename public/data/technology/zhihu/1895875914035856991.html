xparameter - 更智能的 Python 实验参数配置工具
定义实验配置时有几种选择：1）定义数据类，缺点是解析命令行参数需要 argparse 重复定义；2）argparse，缺少自动提示和语法高亮，跨文件引用配置不方便；3）定义配置文件，例如yaml，方便保存配置，缺点是既不能命令行传参，也没有自动提示。基于此，推出 xparameter，一个轻量化的实验参数配置工具。

特性：
✅ 支持语法高亮和成员变量的自动提示
✅ 支持自动绑定 argparse 解析命令行参数
✅ 支持嵌套定义命名空间对同名参数进行隔离
✅ 天然的单例模式，保证跨文件修改的一致性
✅ 支持在赋值时进行基础的变量类型检查
✅ 支持导出 Python dict 和 JSON string

安装：
pip install xparameter

例子：
from xparameter import Configure as C
from xparameter import Field as F

class Config(C):
"""Define your overall description here"""

class checkpoint(C):
"""Define group description here"""
directory = F(str, default="./checkpoints", help="Directory to save model checkpoints")
